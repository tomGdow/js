Introduction

strings
  1. The indexOf() method of String values searches this string and returns the index of the first occurrence of the specified substring
  2. It takes an optional starting position and returns the first occurrence of the specified substring at an index greater than or equal to the specified number
  3. The method returns the index of the first occurrence of the specified substring at a position greater than or equal to position, which defaults to 0
  4. If position is greater than the length of the calling string, the method doesn't search the calling string at all
  5. If position is less than zero, the method behaves as it would if position were 0
arrays
  1. The indexOf() method of Array instances returns the first index at which a given element can be found in the array, or -1 if it is not present
  2. Negative index counts back from the end of the array

Syntax

Strings 
indexOf(searchString)
indexOf(searchString, position) 

// Negative position values interpreted as zero
Arrays 
indexOf(searchElement)
indexOf(searchElement, fromIndex) 

// Negative fromIndex values count from end of array

Compare indexOf and search methods

indexOf
  1. Purpose: Finds the position of a substring in a string
  2. Arguments: Takes a string (not a regex)
  3. Return value: The index (0-based) of the first occurrence, or -1 if not found
search
  1. Purpose: Finds the position of a regular expression match in a string
  2. Arguments: Takes a RegExp (regex), not a plain substring (although you can pass a string, it's converted to a regex)
  3. Return value:The index (0-based) of the first match, or -1 if not found
summary
  1. indexOf(): great for literal substring search; can start searching from a given index
  2. search(): great for pattern matching; uses regex and flags but no start offset
  3. Both return -1 if no match is found
  4. Both return the index of the first match
  5. Both are case-sensitive
  6. Both return 0 for the empty SEARCH
