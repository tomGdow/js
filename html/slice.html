Introduction

string
  1. The slice() method of String values extracts a section of this string and returns it as a new string, without modifying the original string
  2. If indexStart >= str.length, an empty string is returned
array
  1. The slice() method of Array instances returns a shallow copy of a portion of an array into a new array object selected from start to end (end not included) where start and end represent the index of items in that array
  2. The original array will not be modified
  3. The slice() method is a copying method

syntax

string 
slice()
slice(start)
slice(start, end)

// Negative index counts back from the end of the array
// slice() extracts up to but not including end
array 
slice(indexStart)
slice(indexStart, indexEnd)

// Negative index counts back from the end of the array
// slice() extracts up to but not including end

Chat GPT

Compare the array and string slice methods

str.slice(beginIndex, endIndex);
arr.slice(beginIndex, endIndex);
  1. Both accept beginIndex (start position, inclusive)
  2. Both accept endIndex (end position, exclusive; optional)
  4. Both allow negative indices, counting from the end
  5. Non-mutating: Both return a new value without modifying the original
  6. If both parameters are omitted, both return a copy of the entire original value
    1. String: returns the whole string
    2. Array: returns a shallow copy of the entire array
