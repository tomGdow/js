Introduction

  1. In JavaScript, isNaN() is a global function that checks whether a value is NaN (“Not-a-Number”) after attempting to convert it to a number
  2. The function first tries to convert the input to a number
  3. If the result is NaN, it returns true Otherwise, it returns false 

    isNaN(42); // false (42 is a number)

isNaN("42"); // false ("42" converts to 42)

isNaN("Hello"); // true ("Hello" converts to NaN)

isNaN(NaN); // true

isNaN(undefined); // true (undefined → NaN)

isNaN(null); // false (null → 0)

Number.isNaN

  1. isNaN is sometimes too forgiving, because it coerces non-numeric values before checking.
  2. To check for NaN without this coercion, use Number.isNaN() instead
