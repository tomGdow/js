Introduction

    The input event fires when the value of an
    1. input
    2. select
    3. textarea
    element has been changed as a direct result of a user action (such as typing in a textbox or checking a checkbox)
  1. The input event is fired every time the value of the element changes
  2. This is unlike the change event, which only fires when the value is committed, such as by pressing the enter key or selecting a value from a list of options

Code Example (input)

[JavaScript Inserts] 
<label for="myinput">Type</label>

<input type="text" id="myinput" size="40">

<output id="output"></output>
---
const myinput = document.getElementById('myinput');
const myoutput = document.getElementById('output');

myinput.addEventListener('input',
  function(event) {
    myoutput.value=event.target.value
  }
)
