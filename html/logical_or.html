Introduction

  1. The logical OR (||) returns the first truthy value, if any
  2. Otherwise, the last value (if all are falsy) is returned
  3. The logical OR (||) (logical disjunction) operator for a set of operands is true if and only if one or more of its operands is true
  4. It is typically used with boolean (logical) values
  5. When it is, it returns a Boolean value
  6. However, the || operator actually returns the value of one of the specified operands, so if this operator is used with non-Boolean values, it will return a non-Boolean value
  7. It is often used for default values 
    let name = userInput || "Anonymous";

Short-Circuit Evaluation

  1. The logical OR expression is evaluated left to right, it is tested for possible "short-circuit" evaluation using the following rule 
    (some truthy expression) || 
   expr is short-circuit evaluated 
   to the truthy expr
  2. Short circuit means that the expr part above is not evaluated, hence any side effects of doing so do not take effect (e.g., if expr is a function call, the calling never takes place)
