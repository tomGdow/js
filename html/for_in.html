Introduction

  1. The for...in statement iterates over all enumerable string properties of an object (ignoring properties keyed by symbols), including inherited enumerable properties
  2. The loop will iterate over all enumerable properties of the object itself and those the object inherits from its prototype chain (properties of nearer prototypes take precedence over those of prototypes further away from the object in its prototype chain)
  3. A for...in loop only iterates over enumerable, non-symbol properties
  4. Objects created from built-in constructors like Array and Object have inherited non-enumerable properties from Array.prototype and Object.prototype, such as Array's indexOf() method or Object's toString() method, which will not be visited in the for...in loop
  5. Indexes returned by for...in are strings!

Comment: Break and Continue

Break
break stops statement execution and goes to the first statement after the loop
Continue
continue stops statement execution and goes to the next iteration of the loop

Syntax

for (variable in object)
  statement

Object.keys()

  1. Object.keys will return a list of enumerable own string properties
  2. Considers properties attached to the object itself, not its prototype

Object.getOwnPropertyNames()

  1. Object.getOwnPropertyNames will return a list of both enumerable and non-enumerable own string properties
  2. Considers properties attached to the object itself, not its prototype

for(let i in arr) {

  if(typeof arr[i] != "object") {
    console.log(arr[i])

  } else {

    for(let j in arr[i]) {

      if(typeof arr[i][j] != "object") {
        console.log(arr[i][j]);

      } else {

        for(let k in arr[i][j])
          console.log(arr[i][j][k])

      }
    }
  }
}
