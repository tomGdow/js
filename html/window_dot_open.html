Introduction

  1. The open() method of the Window interface loads a specified resource into a new or existing browsing context (that is, a tab, a window, or an iframe) under a specified name

Syntax

open()
open(url)
open(url, target)
open(url, target, windowFeatures)

Description

  1. url (Optional)
    1. A string indicating the URL or path of the resource to be loaded
    2. If an empty string ("") is specified or this parameter is omitted, a blank page is opened into the targeted browsing context
  2. target Optional
    1. A string, without whitespace, specifying the name of the browsing context the resource is being loaded into
    2. If the name doesn't identify an existing context, a new context is created and given the specified name
    3. The special target keywords may also be used:
      1. _self,
      2. _blank (default),
      3. _parent,
      4. _top
  3. windowFeatures (optional)
    1. A string containing a comma-separated list of window features in the form name=value
    2. Boolean values can be set to true using one of:
      1. name
      2. name=yes
      3. name=true,
      4. name=n (where n is any non-zero integer)

Deprecated syntax

window.open(URL, name, specs,replace)
  1. URL (Optional) Specifies the URL of the page to open
  2. name (Optional) Specifies the target attribute or the name of the window
  3. specs (Optional) A comma-separated list of items such as height and width
  4. replace (Optional) Specifies whether the URL creates a new entry or replaces the current entry in the history list
  5. !! But according to W3Schools, the replace argument is DEPRECATED, and Chrome throws an error if this parameter is included

  function openNew() {

    const wind = window.open(
        "", 
        "_blank", 
        "width=300, height=300")

    wind.document.write("<h1>hello</h1>")
  }
  
  mybtn.addEventListener('click', openNew);
