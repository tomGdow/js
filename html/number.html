Introduction

  1. Number values represent floating-point numbers like 37 or -9.25
  2. The Number constructor contains constants and methods for working with numbers
  3. Values of other types can be converted to numbers using the Number() function
  4. A number literal like 37 in JavaScript code is a floating-point value, not an integer
  5. There is no separate integer type in common everyday use
  6. When used as a function, Number(value) converts a string or other value to the Number type
  7. If the value can't be converted, it returns NaN
  8. The JavaScript Number type is a double-precision 64-bit binary format IEEE 754 value
    1. 1 bit for the sign (positive or negative)
    2. 11 bits for the exponent (-1022 to 1023)
    3. 52 bits for the mantissa (representing a number between 0 and 1)
    4. Number = (-1)sign * (1+mantissa) * 2exponent
