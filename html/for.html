Introduction

  1. The for statement creates a loop that consists of three optional expressions, enclosed in parentheses and separated by semicolons, followed by a statement (usually a block statement) to be executed in the loop
  2. All three expressions in the head of the for loop are optional
  3. If omitted, the condition test always evaluates to true
  4. To execute no statement within the loop, use an empty statement (;)

Syntax

for (initialization; condition; afterthought)
  statement

Comment: Break and Continue

Break
break stops statement execution and goes to the first statement after the loop
Continue
continue stops statement execution and
  1. re-evaluates afterthought
  2. then reevaluates condition
