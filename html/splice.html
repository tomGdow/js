Introduction

  1. The splice() method of Array instances changes the contents of an array by removing or replacing existing elements and/or adding new elements in place
  2. The splice() method is a mutating method
  3. To create a new array with a segment removed and/or replaced without mutating the original array, use toSpliced()
  4. To access part of an array without modifying it, see slice()
  5. deleteCount
    1. If deleteCount is omitted, or if its value is greater than or equal to the number of elements after the position specified by start, then all the elements from start to the end of the array will be deleted
    2. However, if you wish to pass any itemN parameter, you should pass Infinity as deleteCount to delete all elements after start, because an explicit undefined gets converted to 0.
    3. If deleteCount is 0 or negative, no elements are removed
    4. In this case, you should specify at least one new element
  6. return value
    1. An array containing the deleted elements.
    2. If only one element is removed, an array of one element is returned.
    3. If no elements are removed, an empty array is returned.

Syntax

splice(start)
splice(start, deleteCount)
splice(start, deleteCount, item1)
splice(start, deleteCount, item1, item2)
splice(start, deleteCount, item1, item2, /* â€¦, */ itemN)
