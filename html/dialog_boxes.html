window.alert()

  1. window.alert() instructs the browser to
    1. display a dialog
    2. include an optional message
    3. wait until the user dismisses the dialog
  2. syntax 
    alert()
alert(message)
    message
    A string you want to display in the alert dialog, or, alternatively, an object that is converted into a string and displayed
  3. The alert dialog should be used for messages which do not require any response on the part of the user, other than the acknowledgement of the message
  4. Dialog boxes are modal windows - they prevent the user from accessing the rest of the program's interface until the dialog box is closed

window.confirm()

  1. window.confirm() instructs the browser to
    1. display a dialog with an optional message
    2. wait until the user either confirms or cancels the dialog
  2. syntax 
    confirm()
confirm(message)
    message
    A string you want to display in the confirmation dialog
  3. return value
    1. true
    2. false

window.prompt()

  1. window.prompt() instructs the browser to
    1. display a dialog prompting the user to input some text
    2. include an optional message
    3. include optional default text in input box
    4. wait until the user either submits the text or cancels the dialog
  2. The submitted value may be assigned to an identifier
  3. If cancel is pressed, null is returned
  4. syntax 
    prompt()
prompt(message)
prompt(message, defaultValue)
    message
    1. A string of text to display to the user
    2. Can be omitted if there is nothing to show in the prompt window
    defaultValue
    A string containing the default value displayed in the text input field.
  5. return value
    1. A string containing the text entered by the user
    2. null
