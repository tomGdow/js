Introduction

  1. The logical AND (&&) (logical conjunction) operator for a set of boolean operands will be true if and only if all the operands are true
  2. Otherwise it will be false
  3. More generally, the operator returns the value of the first falsy operand encountered when evaluating from left to right, or the value of the last operand if they are all truthy
  4. Logical AND (&&) evaluates operands from left to right, returning immediately with the value of the first falsy operand it encounters; if all values are truthy, the value of the last operand is returned
  5. The logical AND expression is a short-circuit operator
  6. As each operand is converted to a boolean, if the result of one conversion is found to be false, the AND operator stops and returns the original value of that falsy operand; it does not evaluate any of the remaining operands
  7. The AND operator has a higher precedence than the OR operator, meaning the && operator is executed before the || operator
